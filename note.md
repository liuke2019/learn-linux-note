## Linux 系统简介

### 终端

在图形界面下，linux提供了终端模拟器（Terminal），本质上对应着linux的 /dev/tty 设备，多用户登录就是通过不同的 /dev/tty 设备完成。

创建一个名为 file 的文件：
```shell
$ touch file
```

进入一个名为 file 的目录：
```shell
$ cd file
```

查看当前所在目录：
```shell
$ pwd
```

### 快捷键

tab：自动补全

ctrl + c：强制终止
